Angular-no-dblclick - Examples

Example

Base controller



Lock Button A
$scope.$broadcast('noDblclick.lock', 'customKey1');

Unlock Button A
$scope.$broadcast('noDblclick.unlock', 'customKey1');

Lock all button in base scope
$scope.$broadcast('noDblclick.lock');

Unlock all button in base scope
$scope.$broadcast('noDblclick.unlock');

Child controller



Lock C
$scope.$broadcast('noDblclick.lock', 'customKey2');

Unlock C
$scope.$broadcast('noDblclick.unlock', 'customKey2');

Lock A
$rootScope.$broadcast('noDblclick.lock', 'customKey1');

Unlock A
$rootScope.$broadcast('noDblclick.unlock', 'customKey1');


Lock all via child scope
$scope.$broadcast('noDblclick.lock');

Unlock all via child scope
$scope.$broadcast('noDblclick.unlock');


Lock all via rootScoope
$rootScope.$broadcast('noDblclick.lock');

Unlock all via rootScoope
$rootScope.$broadcast('noDblclick.unlock');
